# Stock Analysis VBA

Excel VBA Stock Analysis Tool for automated stock data collection, financial ratio analysis, and valuation using the PE method.

---

## Features

- Stock data scraping
- Financial ratio analysis
- PE-based valuation model
- Real-time stock price collection
- Financial statement data extraction

---

## Data Sources

The tool retrieves financial data from public financial information platforms:

- TWSE (Taiwan Stock Exchange)
- MOPS (Market Observation Post System)
- HiStock

---

## Tools

- Microsoft Excel
- VBA (Visual Basic for Applications)

---

## Project Overview

This project builds an automated stock analysis system using Excel VBA.  
It collects financial data, calculates financial ratios, analyzes profit growth, and estimates stock valuation using historical PE ratios and EPS.

The goal is to integrate financial statement analysis with valuation techniques to support stock investment decision making.
